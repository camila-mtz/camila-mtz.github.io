Estructuras de datos que veremos en la materia:
    1. Arreglo: Es una estructura formada por renglones con celdas, cada celda almacena un valor.
      Ejemplo 1
      Ejemplo 2
      Ejemplo 3
    2. Matriz: Es un conjunto bidimensional de números.
    3. Lista: Un arreglo con características especiales para resolver problemas específicos.
    4. Pila: Es un arreglo vertical que trabaja con LIFO: last in first off.
    5. Cola: Es un arreglo horizontal, trabaja con FIFO: first in first out.
    6. Arbol: Estructura de datos con representación de un arbol.
    7. Grafo: Estructura de datos formada por nodos, los nodos están unidos a traves de aristas, estructura de datos mas compleja. Euler
